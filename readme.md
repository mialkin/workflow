# Workflow Core

Showcase of [↑ Workflow Core](https://github.com/danielgerlag/workflow-core) library.

## Prerequisites

- .NET 8 SDK
- Docker

## Running application

```bash
docker-compose --file infrastructure.yaml up --detach
```

```bash
dotnet run --project=src/Workflow.Api
```
